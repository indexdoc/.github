---# IndexDoc 系列开源工具集

IndexDoc 是一套面向办公与开发场景的开源工具合集，聚焦「高效提效、简化流程、隐私安全」三大核心，覆盖文档批量生成、代码标准化生成、本地文档AI交互、法律文书辅助、Markdown编辑五大高频场景，助力个人与团队告别重复劳动，提升工作与开发效率。

本开源项目包含以下核心模块，可独立使用，也可根据需求组合适配各类场景：

## 1. 批量文档助手 (indexdoc-batch-generator)

一款模板驱动的高效文档批量生成工具，专为解决「重复文档编辑、格式不统一」痛点设计，一键即可批量生成符合规范的专业文档，彻底告别繁琐的复制粘贴工作。

### 核心优势

- **高效便捷**：自动化完成文档生成全流程，无需手动编辑替换，大幅缩短文档制作周期，显著提升工作效率；

- **告别重复**：摆脱同类文档反复编辑的繁琐操作，将精力聚焦于核心内容打磨，节省大量时间成本；

- **精准无误**：统一模板与替换规则，避免手动编辑导致的格式错乱、内容遗漏等人为错误，确保文档一致性；

- **灵活适配**：广泛支持法律文书、商务合同、日常报表、办公通知等各类文档类型，适配多行业办公需求。

### 核心功能

支持上传 Excel 数据文件与 Word 模板文件，通过 Excel 定义最终生成文档的文件名规则，以及 Word 模板中需替换的所有字段；在 Word 模板中，仅需将待替换内容以「{列名}」（列名与 Excel 中对应列完全一致）的形式标注，完成简单配置后点击生成按钮，即可快速批量生成所有符合需求的文档，大幅降低手动编辑的时间成本。

## 2. 代码生成器 (indexdoc-model-to-code) / CodeAsst

CodeAsst（编码助理）是一款面向开发团队与个人的高效代码生成工具，核心遵循「数据模型 + 模板 = 代码」的设计理念，致力于实现代码批量生成、标准化输出，减少重复编码工作，提升开发效率与代码质量。

### 核心特性

- **多数据源适配**：支持 PowerDesigner 生成的 PDM 文件导入，可从 Oracle、MySQL、MS SQL Server、PostgreSQL 等多种主流数据库获取表模型，也支持通过 XML 文件手动编辑、定义数据模型；

- **灵活模板语法**：采用类 Velocity（与 Freemarker 语法兼容）风格，降低学习成本，支持自定义模板集；

- **批量高效生成**：一个模板集可包含多个模板文件，一键即可生成一个完整软件项目的所有代码文件与配置文件，无需逐个编写；

- **广泛适用**：适配各类软件公司、开发工作室、研发团队及个人开发者，尤其适合「先设计数据模型、再编写代码」的开发模式。

### 工具价值

数据模型是软件系统的基础，多数系统操作均围绕基础数据的增删查改展开。熟练使用 CodeAsst 后，可批量生成一个系统的几乎所有基础代码；更可基于已有系统的代码模板，快速批量生成多个系统的 Demo 版本，助力开发者快速落地项目、降低开发成本——这并非夸大，而是 CodeAsst 已实现的核心能力。

## 3. 本地文档AI助手 (indexdoc-ai-offline)

一款轻量、高效、隐私优先的本地文档AI交互工具，以客户端形式呈现，核心解决「本地文档精准问答、关键信息提取」需求，无需将文档上传至第三方平台，即可实现人与文档的智能交互，兼顾效率与隐私安全。

### 核心优势

- **自然交互**：无需繁琐操作，以日常对话方式向文档提问，即可快速获取相关信息，上手即用；

- **精准问答**：具备强大的语义理解与内容解析能力，可准确识别用户提问意图，精准提取文档中的关键信息、解读细节、总结核心内容；

- **深度分析**：支持复杂文档的信息提取与内容深度挖掘，可快速梳理文档逻辑、提炼核心观点，节省文档阅读与整理时间；

- **隐私安全**：文件全程本地存储、本地解析，不上传至任何第三方服务器，有效保护文档隐私与数据安全；

- **知识管理**：可结构化整理文档知识，助力个人与企业打造专属知识共享平台，提升知识复用效率。

### 核心功能

支持 PDF、TXT、DOCX 等多种常见文档格式上传，可批量上传多份文件；上传后可自由选中单个文件或文件夹，AI 将仅基于选中的文件内容应答，避免无关信息干扰、提升问答精准度；支持针对选中文件提出个性化问题，快速响应解读需求，同时支持问答历史保存，方便后续回溯、查阅。界面简洁、无需复杂配置，聚焦核心功能，满足各类本地文档交互场景。

## 4. 裁判文书助手 (indexdoc-legal-assistant)

一款专为法律相关从业者设计的文书辅助工具，简化裁判文书制作流程，提升文书生成效率与规范性，适配法院、律所等法律场景的日常办公需求。

### 核心流程

上传「起诉书」及「庭审笔录」（支持 PDF、DOCX 等常见文件格式，提供样例文件供快速参考）→ 工具自动生成判决书初稿 → 支持手动编辑、修改生成的文书内容 → 预览文书效果、导出最终版本，全程简化操作，减少手动撰写工作量。

## 5. Markdown编辑器 (indexdoc-editor)

一款轻量、简洁、适配开发者与办公人群的 Markdown 编辑工具，聚焦「高效编辑、便捷预览、格式规范」核心需求，支持 Markdown 全语法，适配各类 Markdown 文档编辑场景（如项目文档、笔记、README 等）。后续将持续迭代优化，新增更多实用功能（如格式导出、快捷键自定义等），助力用户快速编辑高质量 Markdown 文档。
